Domain

├── Entity
│      داده + رفتارهای مربوط به همان موجودیت
│
├── Value Object
│      اشیای بدون هویت (بعداً با آن آشنا می‌شوی)
│
├── Domain Service
│      قوانین و محاسباتی که بین چند Entity هستند
│
├── Repository (Interface)
│      قرارداد دسترسی به داده
│
└── Validator (در صورت نیاز)
       اعتبارسنجی قوانین دامنه

Application
│
├── Use Case
│      اجرای یک سناریوی کامل
│
├── DTO
│      انتقال داده بین لایه‌ها
│
└── Mapper (در صورت نیاز)
       تبدیل DTO ↔ Entity
